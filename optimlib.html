OptimLib

 

Watch Star Fork

OptimLib is a lightweight C++ library for numerical nonlinear optimization and root finding.

Features
  • C++11 library.
  • Built on the Armadillo C++ linear algebra library.

Author: Keith O'Hara

License: OptimLib is licensed under the GNU General Public License (GPL) version 2, or (at your option) any later version.

Current version:

  • The source code is available on GitHub.

OptimLib can be installed using:

# clone optim
git clone -b master --single-branch https://github.com/kthohr/optim ./optim
# build and install
cd ./optim
./configure
make
make install

The last line will install OptimLib into /usr/local
