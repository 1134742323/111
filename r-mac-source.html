Building R from Source in OSX 10.12.3


This page will guide you through the process of compiling the R-devel on a Mac computer (OSX v10.10.4).

Preliminaries

First we need to install some tools.

  • Get Xcode from the App Store. (The page should look something like this.) New versions of Xcode should come with the command line tools pre-installed.
    • Open a terminal window—search for 'terminal' if you've never needed terminal/bash before—and type: 
      clang++ -v
      you should then see something like 
      Apple LLVM version 8.0.0 (clang-800.0.42.1)
Target: x86_64-apple-darwin16.4.0
Thread model: posix
  • Unfortunately, Xcode does not come with a Fortran compiler, which we will need. The simplest way to obtain an R-compatible version of gfortran is to open a terminal window and type: 
    cd ~/Desktop
curl -O http://r.research.att.com/libs/gfortran-4.8.2-darwin13.tar.bz2
sudo tar fvxz gfortran-4.8.2-darwin13.tar.bz2 -C /
    • The 'sudo' command above will request that you enter your password; this is because we are placing files in what are usually restricted folders (/usr/).
Download and Unpack

First, create an R folder on your Desktop.

cd ~/Desktop
mkdir R

Then download and unpack the latest development version of R:

cd ~/Desktop/R
curl -O ftp://ftp.stat.math.ethz.ch/Software/R/R-devel.tar.gz
tar -zxvf R-devel.tar.gz

We then need to modify the config.site file in the R-devel directory.

#! /bin/sh

CC=clang
CXX=clang++
F77=gfortran-4.8
FC=$F77
OBJC=clang
CFLAGS="-Wall -mtune=core2 -g -O2"
CXXFLAGS="-Wall -mtune=core2 -g -O2"
OBJCFLAGS="-Wall -mtune=core2 -g -O2"
F77FLAGS="-Wall -g -O2"
FCFLAGS=$F77FLAGS

PKG_CONFIG_PATH=/opt/X11/lib/pkgconfig:/usr/local/lib/pkgconfig:/usr/lib/pkgconfig

JAVA_HOME="$(/usr/libexec/java_home)"
JAVA_CPPFLAGS="-I/${JAVA_HOME}/include -I/${JAVA_HOME}/include/darwin"
JAVA_LD_LIBRARY_PATH="${JAVA_HOME}/jre/lib/server"
JAVA_LIBS="-L/${JAVA_HOME}/jre/lib/server -ljvm"
./configure --prefix=/Builds/R-devel --without-x --without-internal-tzcode \
	--with-blas="-framework Accelerate" --with-lapack \
		--enable-R-shlib --enable-memory-profiling

./configure --prefix=/Builds/R-devel --without-x --without-internal-tzcode \
	--with-blas="-lopenblas" --with-lapack \
		--enable-R-shlib --enable-memory-profiling
