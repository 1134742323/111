OptimLib

Watch Star Fork

OptimLib is a C++ library for numerical optimization of nonlinear functions.

Features

  • Parallelized C++11 library of local and global optimization methods, as well as root finding techniques.
  • Advanced derivative-free algorithms, including Differential Evolution (DE) and Particle Swarm Optimization (PSO), with state-of-the-art hybrid metaheuristics.
  • Constrained optimization, from simple box constraints to nonlinear constraints.
  • Built on the Armadillo C++ linear algebra library for fast and efficient matrix-based computation.

Download and Installation

  • The source code is available on GitHub.

OptimLib can be installed in the standard way:

# clone optim
git clone -b master --single-branch https://github.com/kthohr/optim ./optim
# build and install
cd ./optim
./configure
make
make install

The last line will install OptimLib into /usr/local

Configure options:

  • -b dev a 'development' build with install names set to the build directory (as opposed to an install path)
  • -c a coverage build
  • -m specify the BLAS and Lapack libraries to link against; for example, -m "-lopenblas" or -m "-framework Accelerate"
  • -o fast enable aggressive compiler optimization features
  • -p enable parallelization features (using OpenMP)
