StatsLib

 

Watch Star Fork

StatsLib is a templated C++ library for random variable generation and fast computation of statistical distribution functions.

Features
  • Supports the Armadillo C++ linear algebra library for matrix-valued operations.
  • Compile-time computation of density functions, cumulative distribution functions, and quantile functions using C++11 constexpr functions and the GCE-Math library.

Author: Keith O'Hara

Current version:

  • The source code is available on GitHub.

StatsLib is a header-only library, so no installation is required beyond copying the contents of the include folder.

To build the test files:

# clone stats
git clone -b master --single-branch https://github.com/kthohr/stats ./stats
# compile tests
cd ./stats/tests
./cov_setup
