GCE-Math: Gaussian error function

Definition: 
 
constexpr long double erf(const long double x)

Details:

The Gaussian error function:

$$\text{erf}(x) = \frac{2}{\sqrt{\pi}} \int_0^x \exp( - t^2) dt$$

is computed using a continued fraction representation: $$\text{erf}(x) = \frac{2x}{\sqrt{\pi}} \exp(-x^2) \dfrac{1}{1 - 2x^2 + \dfrac{4x^2}{3 - 2x^2 + \dfrac{8x^2}{5 - 2x^2 + \dfrac{12x^2}{7 - 2x^2 + \ddots}}}}$$
