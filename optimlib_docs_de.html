OptimLib: Differential Evolution

Function arguments:

  • init_out_vals a column vector of initial values; will contain the final values.
  • opt_objfn the function to be minimized, taking three arguments:
    • vals_inp a vector of inputs;
    • grad_out an empty vector, as DE does not require a gradient vector as input; and
    • opt_data additional parameters passed to the function.
  • opt_data additional parameters passed to the function.
  • opt_params parameters controlling the optimization routine; see below.

Optimization control parameters:

  • int de_n_gen number of vectors to generate.
  • int de_check_freq how many generations between convergence checks.
  • double de_par_F the value $F$ in the details section below.
  • double de_par_CR the value $CR$ in the details section below.

Details:

Let $x_{i}$ denote the values at stage $i$ of the algorithm. The mutation step:

$$X^* = X_i(c,:) + F \times (X_i(a,:) - X_i(b,:))$$

The crossover step:

$$X_c^* (j,k) = \begin{cases} X^*(j,k) & \text{ if } u_k \leq CR \text{ or } k = r_k \\ X_i (j,k) & \text{ else } \end{cases}$$

The update step:

$$X_{i+1} (j,:) = \begin{cases} X_c^*(j,:) & \text{ if } f(X_c^*(j)) < f(X_i(j)) \\ X_i (j) & \text{ else } \end{cases}$$

The algorithm stops when the relative improvement in $f$ is less than err_tol, or the total number of 'generations' exceeds a desired (or default) value.

Example

In this example, we consider minimization of the Rastrigin function:

$$\min_{x \in [-5,5]^n} \left\{ A n + \sum_{i=1}^n \left( x_i^2 - A \cos(2 \pi x_i) \right) \right\}$$

This is a well-known test function for numerical optimization software, as the objective function is 'bumpy' and contains many local minima.

A contour plot of the two dimensional problem, with $A = 10$, is given below.

The solution to the minimization problem is the zero vector. Code implementing this in OptimLib is given below.

#include "optim.hpp"

struct rastrigin_fn_data {
    double A;
};

double
rastrigin_fn(const arma::vec& vals_inp, arma::vec* grad_out, void* opt_data)
{
    const int n = vals_inp.n_elem;

    rastrigin_fn_data* objfn_data = reinterpret_cast<rastrigin_fn_data*>(opt_data);
    const double A = objfn_data->A;

    double obj_val = A*n + arma::accu( arma::pow(vals_inp,2) - A*arma::cos(2*arma::datum::pi*vals_inp) );
    //
    if (grad_out) {
        *grad_out = 2*vals_inp + A*2*arma::datum::pi*arma::sin(2*arma::datum::pi*vals_inp);
    }
    //
    return obj_val;
}

int main()
{
    rastrigin_fn_data test_data;
    test_data.A = 10;

    arma::vec x = arma::ones(2,1);

    bool success = optim::de(x,rastrigin_fn,&test_data);

    if (success) {
        std::cout << "de: test completed successfully." << std::endl;
    } else {
        std::cout << "de: test completed unsuccessfully." << std::endl;
    }

    arma::cout << "de: solution to test:\n" << x << arma::endl;

    return 0;
}
